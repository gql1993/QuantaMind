# QuantaMind 量智大脑 —— 产品与技术介绍白皮书

## 1. 产品概述

**QuantaMind（量智大脑）** 是一个专门面向量子科技领域的**自主科研人工智能中台 (Quantum Technology Autonomous Research AI Platform)**。

其核心目标是通过结合大语言模型 (LLM) 的逻辑推理能力与量子计算、工业制造领域的专业工具软件，打造一个超级 AI 科学家团队。该平台能够覆盖从**超导量子芯片设计、电磁仿真、理论物理分析、微纳制造工艺、量子测控到数据分析**的全生命周期，实现研发过程的智能化、自动化闭环。

---

## 2. 核心系统架构 (AI 中台引擎)

QuantaMind 在架构上采用了模块化的智能体底座设计，包含以下六大核心组件：

1. **Gateway (网关层)**: 基于 FastAPI + WebSocket 构建。负责客户端（CLI 命令行、Web 浏览器端）的请求接入与路由，支持高并发 REST API 与流式 (Streaming) 的实时对话体验。
2. **Brain (大模型大脑)**: 逻辑推理和决策的中枢。原生支持并优化了本地部署的 LLM（默认配置为 `Ollama` + `qwen2.5:7b`），确保敏感量子科研数据的私密性与零 API 成本，同时也支持扩展主流的 OpenAI 格式模型接口。
3. **Hands (执行工具库)**: AI 与物理世界/数字世界交互的“双手”。内部封装了数十个甚至上百个 `Tool Calls`，允许 AI 自主调用底层硬件（如 ARTIQ）、工业软件（如 Ansys、Qiskit）及数据中心接口。
4. **Memory (项目记忆)**: 采用 Markdown 结合本地文件存储的长期记忆机制。它为不同的研发项目隔离出独立上下文，使得跨周期的科研对话能够保持连贯。
5. **Heartbeat (心跳/自驱循环)**: 后台定时与自主触发机制。通过定时任务（如 `INTEL_SCHEDULE_HOUR`），平台可以脱离用户直接指令，自动执行文献追踪、预警巡查、摘要生成和信息推送。
6. **Skills (技能库)**: 结合 YAML 配置与 Markdown 提示词构建的动态扩展层，研发人员可以随时为系统挂载新的领域技能而无需修改底层核心代码。

## 3. 多智能体专家系统 (Multi-Agent System)

QuantaMind 并非一个单体对话助手，而是一支虚拟的“科研团队”。通过 `Orchestrator`（协调者中枢），系统能够通过意图识别和语义路由，自动将用户的复杂问题分发给最合适的**专家智能体 (Agent)**：

| 专家智能体 | 核心职责与能力描述 | 对接工具与技术栈 |
| :--- | :--- | :--- |
| 🧑‍🎨 **芯片设计师** <br>(Chip Designer) | 精通超导量子芯片（如 transmon）全流程版图设计、组件布线与物理设计导出。 | Qiskit Metal, KQCircuits, qEDA, GDS 导出 |
| 🧙‍♂️ **理论物理学家** <br>(Theorist) | 作为理论决策中枢，负责哈密顿量建模 (EPR/BBQ)、噪声误差预算、测控参数优化与物理故障归因。 | 内部理论计算引擎，知识检索 |
| 🏭 **制造工程师** <br>(Process Engineer) | 统筹制造团队，下设工艺、缺陷、良率子视角；监控晶圆与批次流片，追踪 SPC 及设备告警。 | ChipMES, OpenMES, SECS/GEM, Grafana |
| 📡 **测控科学家** <br>(Measure Scientist) | 精通量子比特表征与性能优化，处理 T1/T2/Rabi 谱，执行校准测试及错误缓解。 | ARTIQ, Qiskit Pulse, Mitiq |
| 💻 **仿真工程师** <br>(Simulation Engineer) | 负责电磁仿真全生命周期，包括本征模/驱动模仿真、电容矩阵提取、LOM/EPR 分析。 | Ansys HFSS, Q3D (经仿真组件适配) |
| 📊 **数据分析师** <br>(Data Analyst) | 负责数据中台分析，执行跨域数据查询、数据流调度及 Text2SQL 自然语言取数分析。 | Apache Doris, SeaTunnel, QData |
| 🕵️ **情报与知识专家** <br>(Intel & Knowledge)| 持续跟踪 arXiv 最新量子计算论文，生成趋势情报与日报推送；维护项目文档与知识图谱。 | 飞书/企微 Webhook, PGVector, ArXiv API |
| 🔧 **设备与算法研发** <br>(Ops & Algorithm) | 测控实验室硬件运维、底层通信指令下发；探索量子算法 (VQE/QAOA) 与电路编译转译。 | Qiskit, ARTIQ 设备接口 |

## 4. 关键技术栈与生态集成

QuantaMind 集成了大量先进的 IT 技术与量子领域专用工具，实现了真正的跨域协同：

- **基础与 AI 框架**: `Python`、`FastAPI`、`Ollama`、`PGVector` (向量数据库，支持 RAG 知识检索)。
- **大数据与工业 IoT**: 
  - `Apache Doris` (OLAP 实时数据仓库分析)
  - `Apache SeaTunnel` (跨数据源高速 ETL 同步)
  - `Grafana` (工业监控与看板)
  - `SECS/GEM` (半导体设备通信标准)
- **数据库与存储阵列**: `PostgreSQL` (业务与知识库)、`SQL Server` (兼容传统 MES)、`MinIO` (大规模实验对象数据存储)。
- **量子专业软件库**: `Qiskit (Metal / Pulse)`、`KQCircuits`、`qEDA`、`Mitiq` (量子错误缓解)、`QCoDeS` (测量自动化)。

## 5. 典型业务应用场景

### 场景一：端到端量子芯片敏捷设计与打样验证
用户只需用自然语言描述：“帮我生成一个 20 比特的 transmon 芯片基础版图，采用 SQUID 结”。
**芯片设计师 Agent** 会调用 Qiskit Metal 自动生成对应的组件及布线代码并导出 GDS。接着 **仿真工程师 Agent** 无缝介入，调用 HFSS 计算寄生电容和本征频率。如果不满足设计要求，**理论物理学家 Agent** 将介入分析误差原因并提出参数修正建议，形成“设计-仿真-评估”的闭环迭代。

### 场景二：全自动全球前沿科研情报监听
依托于 **Heartbeat** 机制，**情报员 Agent (Intel Officer)** 每日凌晨会在后台自动检索 arXiv 等预印本网站关于“量子纠错”、“超导芯片测控”的最新论文。系统自动阅读摘要、分析技术趋势与突破点，形成带有核心参数萃取的《每日量子技术简报》，并通过飞书或企业微信 Webhook 准时推送给相关研发群，同时将重要知识点沉淀到 PGVector 向量数据库中供未来查阅。

### 场景三：跨域(产线-测控)良率异常排查
当**设备与制造工程师 Agent** 通过 MES 接口捕获到某批次晶圆存在特定区域的良率下降告警时，可交由 **数据分析师 Agent** 将制程工艺参数与下游测控实验室传回的比特 T1/T2 数据进行关联。通过 Doris 引擎的查询分析，结合**理论物理学家 Agent** 的根因诊断模型（故障树推理），能够迅速定位到可能引发损耗的特定湿法刻蚀工序，极大缩短排障周期。
