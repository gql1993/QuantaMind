# QuantaMind 产品原型（可点击）

本目录为 **QuantaMind 量智大脑客户端** 的产品原型，供产品评审与前端开发参考。

## 使用方式

- 用浏览器直接打开 **`index.html`**（建议 Chrome / Edge）。
- 左侧导航可切换：**对话**、**任务**、**自主发现**、**技能**、**设置**。
- 原型为静态展示，输入与按钮无后端逻辑，仅用于界面与流程演示。
